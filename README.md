# Резервное копирование изображений с сайтов про животных

Курсовой проект на Python для резервного копирования изображений с сайтов про кошек и собак в облачное хранилище Яндекс.Диск.

## Описание проекта

Проект предоставляет две основные функции:
1. Загрузка изображений кошек с текстом с сайта cataas.com
2. Загрузка изображений собак различных пород с сайта dog.ceo

Все изображения загружаются напрямую в облачное хранилище Яндекс.Диск без сохранения на локальный диск.

## Функциональность

### Для кошек (cataas.com)
- Генерация изображений кошек с пользовательским текстом
- Создание папки на Яндекс.Диске с названием группы
- Сохранение изображений с именем, соответствующим тексту на картинке
- Возможность загрузки нескольких изображений за один запуск
- Сохранение метаинформации о загруженных файлах в JSON-файл

### Для собак (dog.ceo)
- Загрузка изображений собак указанной породы
- Автоматическое определение и обработка подпород
- Создание папки на Яндекс.Диске с названием породы
- Возможность указания количества изображений для каждой породы/подпороды
- Сохранение метаинформации о загруженных файлах в JSON-файл

## Технологии

- Python 3.8+
- Requests - для HTTP-запросов
- TQDM - для отображения прогресса загрузки
- Яндекс.Диск REST API - для работы с облачным хранилищем

## Установка и запуск

1. Клонируйте репозиторий:
```bash
git clone <URL вашего репозитория>
cd <имя папки проекта>
```
2. Установите зависимости:
```bash
pip install -r requirements.txt
```
3. Запустите программу:
```bash
python cat_and_dog.py
```
## Использование

При запуске программы выберите задание:

- 1 - для работы с изображениями кошек
- 2 - для работы с изображениями собак
- 3 - для выхода из программы

Для работы с Яндекс.Диском потребуется:

- Токен OAuth-авторизации (можно получить на [Полигоне Яндекс.Диска](https://yandex.ru/dev/disk/poligon/))
- Название группы (для кошек) или породы собаки (для собак)

Программа предоставит интерактивный интерфейс для ввода всех необходимых параметров.

## Примеры работы

### Загрузка изображений кошек
Введите токен для Яндекс.Диска: ваш_токен_здесь

Введите название группы: PD-134

Введите текст для картинки с котиком: Hello World

Сколько изображений загрузить? (по умолчанию 1): 3



### Загрузка изображений собак
Введите токен для Яндекс.Диска: ваш_токен_здесь

Введите породу собаки: retriever

Сколько изображений для каждой породы/подпороды загрузить?

(по умолчанию 1): 2

## Возможные ошибки и решения

### Ошибка аутентификации Яндекс.Диска
- Проверьте правильность введенного токена
- Убедитесь, что токен имеет необходимые права доступа

### Проблемы с сетевым соединением
- Проверьте подключение к интернету
- Попробуйте использовать VPN при блокировке доменов

### Ошибки API
- Породу собак можно вводить только на английском языке
- Некоторые породы собак могут отсутствовать в базе dog.ceo
- API cataas.com может быть временно недоступно
